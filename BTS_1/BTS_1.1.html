1) A cosa serve Bootstrap?

E' un framework frontend gratuito e facile per l'utilizzo per la gestione del css in maniera semplice e veloce da configurare.

2) Cosa devi importare per creare una pagina web con Bootstrap?

CSS e JS bundle (include Popper). Esempio CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>
3) Cos’è un CDN? Vantaggi?

E' un modo per importare le librerie senza doverli scaricare; il vantaggio è che si può facilmente cambiare versione direttamente dal tag senza dover riscaricare tutto da capo.

4) A cosa serve content="width=device-width, initial-scale=1"?

Imposta la larghezza uguale al dispositivo e zoom iniziale 1 per abilitare il layout responsive.

5) Container vs Container Fluid

Container normale ha una dimensione preimpostata, mentre il fluido dà la possibilità di cambiare dimensione in base alla dimensione dello schermo disponbibile. Prova: BTS_1.1.1.html
